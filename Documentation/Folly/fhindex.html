Folly(verse) Hub

Folly Hub is the front-end interface for Folly Server and exported Tames' projects. You can download the source code from here and its runnable Windows app from here.

Importing projects

When you open Folly Hub, it automatically looks for all the projects in its Projects folder. If there are ZIP files in this folder that do not match a folder, it will ask you if you want to extract and import them (see the image below). You can select which one you would like to import. After they are imported, they will appear on the projects list.

You can also import projects from other locations by clicking on either Import Files (where you select ZIP files) or Import Folder. The imported projects will appear on the projects list. Please note that these are projects exported by Tames , not the folders or ZIP files intended for WebGL platform.

Running projects

For each project, there is a Run button at its bottom-right corner. When you click on it, a dropdown menu appears with the following options:

  • Offline: the project will be opened without connection to the internet.
  • As server: the project will be opened as the local server. Once this is clicked, a window will appear that shows your computer's local IP address and required ports. You need to give these values to other user (aka. "clients") wishing to connect to this project (see the next option).
  • As client: the project will be opened as a client. It will ask you to enter the IP address and ports for the server which is given to you by the person who has started as server (see above) or the already running remote server.
